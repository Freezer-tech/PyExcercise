In questo readme troverete tutte le informazione che vi servono per poter svolgere gli esercizi.
Saranno strutturati in modo tale che per poter svolgere l'esercizio successivo, dovrete saper utilizzare ed
    eseguire gli esercizi precedenti in modo corretto. 
Le diverse tipologie di esercizi saranno suddivisi in "Progetti" come ad esempio Py00, Py01 etc...

I Progetti saranno suddivi in questo modo:
    - Py00 --> Tipo di Dati
    - Py01 --> Defizione di una funzione e Induzione
    - Py02 --> Istruzioni Condizionali e Cicli ( While e For )
    - Py03 --> Dizionari e Liste (coming soon)
    - coming soon ...

L'appredimento sarà tra di voi. Io potrò darvi dei consigli ma non la soluzione. Se non sai una cosa, chiedila ad uno
    dei tuoi compagni, sennò cerca su Google !!

NB !!!!!!!!!

Per testare le vostre funzioni, all'interno dei vostri progetti c'è un file chiamato tester.py. Per eseguirlo, dovete
installare la libreria 'pytest'.

Per installare pytest su Windows, puoi utilizzare il gestore di pacchetti pip.

    1. Assicurati di avere Python installato sul tuo sistema. Se non hai ancora installato Python, puoi farlo da qui: https://www.python.org/downloads/

    2. Apri il prompt dei comandi di Windows (cmd).

    3. Verifica che pip sia già installato digitando il comando:

        pip --version


    Se non hai ancora installato pip, dovrai installarlo prima di poter installare pytest. Per installare pip, digitare il comando:

        python -m ensurepip --upgrade

    Una volta che hai verificato che pip sia installato, puoi installare pytest digitando il comando:

        pip install pytest


Qunando tutto sarà installato, dal vostro terminale muovetevi con 'cd' all'interno della cartella dove avete scaricato i progetti.
Entrate nel progetto da testare (es. Py00) e nel terminale eseguite questo comando:

        python tester.py

Una volta eseguito, dovreste essere in grado di vedere quali test avete passato e quelli che NON passati. Quelli NON passati saranno colorati
    di rosso e potrete vedere il risultato aspettato e quello che invece l vostra funzione a generto.
